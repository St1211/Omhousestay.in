📸 Photo Uploader

Upload images to your GitHub repository
📤

Click or drag & drop to upload photos

Supports: JPG, PNG, GIF, WEBP (Max 10MB each)

📋 How to use:
1. Select photos (click or drag)
2. Click "Upload to GitHub"
3. You'll get a code to add to your website

⚠️ Important: This tool prepares your photos for upload. You'll still need to:
  • Save the generated code
  • Manually add files to your GitHub repository
  • Or use the GitHub API (requires token - see advanced mode)
